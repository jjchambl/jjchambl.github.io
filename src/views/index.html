The Loan Ranger

The Loan Ranger was developed during the Magic City Coders program in collaboration with BBVA Compass, S|ngular, UAB, and Innovation Depot. It will be presented at Birmingham's Innovation Week in August (date and time TBD). Using Google's Polymer framework and working under a SCRUM methodology, I developed the Loan Ranger to serve as a debt reduction tool for bank customers wanting to take more proactive control of their loans. The application interacts with a data model I built using Firebase, and returns a hypothetical list of three loans listed by amount (located in the Setup tab in the navbar). The user can click/tap to select the loan to target, and will then specify the amount that they want applied for each swipe of a (again hypothetical) debit card; the default value rounds up every purchase to the nearest dollar amount and applies the difference towards the targeted loan; the user can choose to add dollar increments per transaction up to a maximum of $20 per swipe. After completing preferences, the Account Detail view shows the user's overall progress to paying off their loan based off a month of card transactions.
